## 📁 リポジトリの構成

### templates/
- othello_template/ - オセロゲームのテンプレート 
- 2048_template/ - 2048のテンプレート
- music_app/ - 音楽配信アプリのテンプレート

### projects/ 
- othello/ - オセロの完成コード
- 2048/ - 2048の完成コード
- soundscape_odyssey/ - 音楽配信アプリの完成コード 

### sample_data/
- サンプルの音声、画像ファイル
